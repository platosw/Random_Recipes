# **Random Meal Recipes**

A simple website that displays random meal random recipe data using the mealdb.com API.

<br>

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- Skeleton CSS
- Google Fonts
- themealdb.com API
